# DESIGN.md

## 气质与意象
- 保险柜开锁、密室破解、暗夜侦探。深色金属质感背景，数字排列如密码锁转盘，金色琥珀光点缀关键信息。空气中弥漫紧张而专注的推理氛围。

## 配色方案
- 背景：深炭黑 `#0f1117` ~ `#1a1d27`
- 卡片/面板：深灰 `#1e2130` 带微透明
- 主强调色：琥珀金 `#f59e0b`（★ 反馈、按钮高亮）
- 次强调色：银白 `#e2e8f0`（☆ 反馈、辅助文字）
- 正确/成功：翡翠绿 `#10b981`
- 错误/失败：珊瑚红 `#ef4444`
- 数字显示：等宽字体，琥珀金色

## 字体排版
- 数字/密码：等宽字体 `font-mono`，大号加粗，赋予机械感
- UI 文案：系统无衬线字体
- 标题：大号加粗，紧凑字距

## 动效与交互
- 数字输入：逐位翻转/弹入动画
- 提交猜测：按钮按下时微缩 + 光晕扩散
- ★/☆ 反馈：逐个浮现，带缩放弹跳
- 猜测历史：滑入列表，从下方推入
- 成功破解：金色粒子爆发 + 数字闪烁

## 页面结构
- 顶部：标题 + 模式切换（游戏模式 / 解题助手）
- 中部左侧：猜测历史列表（每次猜测 → 反馈 ★☆）
- 中部右侧：输入区域 / 推荐猜测面板
- 底部：规则说明折叠区

## 设计禁忌
- 禁止科技蓝/蓝紫色渐变
- 禁止卡通/萌系风格
- 禁止过多纯白大面积色块
- 禁止花哨背景干扰数字阅读
